# Personal Website

Portfolio website for Olivier Halupczok - GenAI Engineer.

## Tech Stack

- **Astro** - Static site generator
- **Preact** - Interactive components (~4kb)
- **CSS** - Glassmorphism design with motion animations

## Features

- Critical CSS inlined for fast first paint (<14kb)
- Contact form with Formspree
- Calendly booking widget
- Vercel Analytics with event tracking
- Responsive design

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
